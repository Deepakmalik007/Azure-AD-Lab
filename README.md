# Azure AD Lab

Hands-on lab for learning **Azure Active Directory and Microsoft 365 administration**.

## Topics Covered
- Azure Active Directory
- User and group management
- Role-based access control
- Microsoft 365 administration
- Identity and access management

## Technologies Used
- Microsoft Azure
- Azure Active Directory
- Microsoft 365
- PowerShell

## Goal
This lab demonstrates real-world cloud identity and access management tasks commonly performed by IT support engineers and cloud administrators.
